# dotfiles for fun and not so much profit

My feeble attempt to take a dotfiles project (which is awesome) ...and ruin it.

This is still a fork of [@mathias's dotfiles repo](https://github.com/mathiasbynens/dotfiles),
but is slowly but surely diverging.

Hat tip to @joshmock and [his dotfiles](https://github.com/joshmock/dotfiles)
as well.

## Notes to self

Use [`dotfiles`](https://pypi.org/project/dotfiles/) to manage this mess.

Install [`ohmyzsh`](https://github.com/ohmyzsh/ohmyzsh#basic-installation) as well.

And [`powerlevel10k`](https://github.com/romkatv/powerlevel10k#getting-started).

Don't forget [`brew`](https://brew.sh/), and then:

```
brew install hub
brew install macvim
brew install pyenv
brew install z
```
